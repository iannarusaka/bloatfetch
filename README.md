# bloatfetch

neofetch for windows users

it scans your machine for junk you never asked for.
and gives you a score based on how much you care for your own user experience.

**it removes nothing.** 
just mocks you

## install

with scoop:

```powershell
scoop bucket add iannarusaka https://github.com/iannarusaka/scoop-bucket
scoop install bloatfetch
```

or with cargo:

```sh
cargo install --git https://github.com/iannarusaka/bloatfetch
```

then run:

```sh
bloatfetch
```

if you dislike the roast, debloat yourself.
